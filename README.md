# ZeloPet - Cuidado e Mobilidade Pet

Plataforma integrada de serviços pet para Governador Valadares, unindo a confiança de cuidadores especializados à logística de transporte pet seguro.

## Tecnologias
- **React 19**
- **Vite 7**
- **Tailwind CSS 4**

## Funcionalidades
- **Explorar Passeadores:** Marketplace de profissionais verificados.
- **PetDriver (Uber Pet):** Transporte especializado com veículos adaptados.
- **Assinaturas:** Planos mensais com descontos progressivos.

## Como Executar
1. Instale as dependências: `npm install`
2. Inicie o servidor de desenvolvimento: `npm run dev`
